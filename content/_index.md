このサイトは、oTree で経済実験用のプログラムを作成するための総合ガイドです。
初心者から中級者まで、段階的に学べる構成になっています。

{{% notice style="info" title="対象バージョン" %}}
このサイトは **oTree 6.x** を基準にしています。
oTree 5.x との違いは各ページで注記しています。
{{% /notice %}}

## インストール

oTree は [PyPI](https://pypi.org/project/otree/) から `pip` でインストールできます。

```bash
pip install otree
```

詳細なインストール方法やオプションについては、[oTree 公式ドキュメント](https://otree.readthedocs.io/ja/latest/install.html)を参照してください。
